# Handwritten-Character_recognition

Recognize handwritten digits using Deep Learning and TensorFlow.

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Installation

pip install -r requirements.txt

## Run

python handwritten_recognition.py

## Features
- Image preprocessing
- Digit recognition
- Deep learning model
- Accuracy evaluation
